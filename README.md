# My CV Website

A clean, responsive personal CV website built with only HTML and CSS.

## Features
- Header with name, title, and photo
- About Me, Education, Skills, Experience/Projects, Contact, Footer
- Responsive design (mobile & desktop)
- Flexbox layout
- Consistent colors and fonts
- No frameworks or JS

## Technologies Used
- HTML5
- CSS3 (Flexbox)
